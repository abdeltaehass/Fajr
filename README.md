# Fajr

An Islamic prayer times app built with Flutter. Displays accurate salah times based on your location with a beautiful Islamic-themed UI.

## Features

- All 5 daily prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha)
- Live countdown to the next prayer
- Hijri and Gregorian date display
- Islamic-themed UI with green and gold color scheme
- GPS-based location detection
- Pull-to-refresh support

## Getting Started

```bash
flutter pub get
flutter run
```

## API

Prayer times are fetched from the [Aladhan API](https://aladhan.com/prayer-times-api) using the ISNA calculation method.
